# 対象システム仕様メモ: アット葬儀 (funeral.at-sougi.com)

調査日: 2026-06-28 / 出典: Web検索（公開情報）+ at-sougi.com 公式

## サービス概要
葬儀社向けのオンライン葬儀プラットフォーム。喪主・遺族が訃報をオンラインで案内し、
参列できない人もオンラインで弔意を示せる（祭壇閲覧・香典決済・供花供物注文・メッセージ）。

## コア機能（クローン対象）
| # | 機能 | 概要 |
|---|------|------|
| 1 | オンライン訃報案内 | 故人情報・葬儀日程・会場・喪主情報をページ化。URL/メール/SNSで共有。受信者は案内確認。 |
| 2 | オンライン祭壇 | 祭壇・会場・故人の思い出を静止画/動画で公開。バーチャル参拝（焼香/献花）。 |
| 3 | オンライン香典決済 | 参列者がクレジットカードで香典を決済。喪主へ送金。現金管理・集計不要。 |
| 4 | メッセージ（お悔やみ） | 喪主・遺族へメッセージ+画像を送信。 |
| 5 | 供花・供物オンライン注文 | 訃報受信者から供花/供物を直接注文・決済。 |

## ユーザーロール（想定）
- **葬儀社管理者** — 案件作成・管理、テンプレート設定。
- **喪主/遺族** — 訃報作成・公開範囲設定、香典/メッセージ受領、入金確認。
- **参列者（訪問者）** — 訃報閲覧、香典決済、供花供物注文、メッセージ送信、祭壇参拝。

## ログインフロー
- `/account/sign-in` にメール+パスワードでログイン（喪主/葬儀社側の管理画面）。
